# Web-Application-Project
This project was developed as part of a university group project in the Wirtschaftsinformatik program.

## Technologies
- Java
- Apache Tomcat
- H2 Database
- HTML
- Eclipse IDE
- JavaScript

## Description
The project is a web application that runs on a local server using Apache Tomcat. 
It demonstrates backend logic, database integration and web interface development.
